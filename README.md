# CRUD Simples - Trabalho UFU

Este projeto consiste em um sistema CRUD (Create, Read, Update e Delete) desenvolvido em Python para fins acadêmicos na Universidade Federal de Uberlândia (UFU).
O sistema permite o cadastro, consulta, atualização e remoção de ativos, utilizando um banco de dados SQLite para armazenamento das informações.

# Tecnologias Utilizadas
Python 3.x
SQLite3
Tkinter (Interface Gráfica)

Como Executar

O arquivo principal do projeto é o ```index.py```.

Ao executar o programa, o usuário poderá escolher entre:

Modo CLI (Command Line Interface)
Execução diretamente pelo terminal, utilizando menus e comandos de texto.
Modo Interface Gráfica (GUI)
Execução através de uma janela gráfica desenvolvida com Tkinter.
Executando o projeto
`python index.py`

Após iniciar, selecione a opção desejada para utilizar o sistema.

# Dependências
Tkinter

O Tkinter é utilizado para a interface gráfica da aplicação.

Ubuntu/Debian
`sudo apt update`
`sudo apt install python3-tk`

# Objetivo Acadêmico

Este projeto foi desenvolvido como atividade acadêmica da Universidade Federal de Uberlândia (UFU), com o objetivo de praticar conceitos de:

Programação em Python
Manipulação de banco de dados SQLite
Desenvolvimento de interfaces gráficas com Tkinter
Implementação de operações CRUD
Organização e estruturação de projetos
